# Rotom Stero Spotify Display
My Spotify display was based on the pokemon Rotom and one of his types being stereo, which fit the theme of music as it was a speaker based pokemon. This project will get your Spotify track and display the music on to a display screen. I personally chose this project because it looked like a cool thing to do for my first project and was a starter project for Stasis.
